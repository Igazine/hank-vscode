# Hank for VSCode

The official Visual Studio Code extension for the **Hank** symbolic orchestration language.

Hank is a purely symbolic, instruction-oriented embeddable language designed for secure, dynamic automation. This extension provides a professional-grade development environment for writing and debugging Hank scripts.

## Features

-   **Intelligent Autocomplete**: Context-aware suggestions for all native tasks and custom identifiers.
-   **Rich Hover Documentation**: Instant access to task signatures, descriptions, and usage examples.
-   **Deep Navigation**: Full "Go to Definition" support for symbols and macros, including resolution across file includes.
-   **Semantic Insights**: Real-time feedback on language-specific behaviors like shadowing and "evaluate-then-bind" logic.
-   **Embedded Execution**: Run Hank scripts directly within VSCode via the "Play" button with live output streaming.
-   **Syntax Highlighting**: Precise, color-coded representation of Hank's unique symbolic grammar.

## Getting Started

1.  Open any file with a `.hank` extension.
2.  The extension will activate automatically and start providing language services.
3.  Click the **Play** button in the editor title bar to execute your script using the built-in runner.
